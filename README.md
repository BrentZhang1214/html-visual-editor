# HTML Visual Editor / HTML可视化编辑器

[English](#english) | [中文](#chinese)

---

<a name="english"></a>
## English

A visual HTML editor that works like editing a Word document. No installation required - just open in browser and start editing!

### Features

- **Visual Editing**: Edit HTML content visually like a Word document
- **Source Code View**: Toggle between visual and HTML source code view
- **Style Isolation**: Uses iframe to prevent style conflicts between editor and target HTML
- **Bilingual Support**: Chinese/English language toggle
- **Single File**: All in one HTML file - no dependencies

### Usage

1. Download `html-editor.html`
2. Open in any modern browser (Chrome, Firefox, Edge, Safari)
3. Click "Open HTML File" to load an HTML file
4. Edit visually or toggle to source code view
5. Click "Save" to download the modified file

### Technical Highlights

- **iframe Style Isolation**: Prevents editor styles from affecting the target HTML
- **contentEditable**: Native browser editing capability
- **File API**: Read and save local files without server
- **Google Fonts**: Clean typography with modern fonts

## 中文

一个像编辑Word文档一样的HTML可视化编辑器。无需安装，浏览器直接打开即可使用！

### 功能特点

- **可视化编辑**：像编辑Word文档一样编辑HTML内容
- **源码视图**：可视化与HTML源码一键切换
- **样式隔离**：使用iframe防止编辑器样式与目标HTML冲突
- **双语支持**：中英文一键切换
- **单文件实现**：所有功能集成在一个HTML文件中，无依赖

### 使用方法

1. 下载 `html-editor.html`
2. 在任意现代浏览器中打开（Chrome、Firefox、Edge、Safari）
3. 点击"打开HTML文件"加载HTML文件
4. 可视化编辑或切换到源码视图
5. 点击"保存"下载修改后的文件

### 技术亮点

- **iframe样式隔离**：防止编辑器样式影响目标HTML
- **contentEditable**：浏览器原生编辑能力
- **File API**：无需服务器即可读写本地文件
- **Google Fonts**：现代字体排版
